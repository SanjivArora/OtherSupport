# Description

This template deploys a fully parameterized Storage account, of the "kind" "StorageV2"

## Specs

It defaults to the SKU "Standard_LRS", can be changed at runtime. 

### Allowed SKUs
- "Standard_LRS", 
- "Standard_GRS", 
- "Standard_RAGRS", 
- "Standard_ZRS", 
- "Premium_LRS", 
- "Premium_ZRS", 
- "Standard_GZRS", 
- "Standard_RAGZRS"
